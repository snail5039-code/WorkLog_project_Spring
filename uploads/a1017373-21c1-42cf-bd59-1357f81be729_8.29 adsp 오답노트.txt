오답 노트
* 데이터웨어하우스느느 기업 내의 의사결정지원 애플리케이션에 정보 기반을 제공하는 하나의
통합된 데이터 저장 공간을 말한다. 다음중 틀린것은?
- 데이터웨어하우스에서는 데이터의 지속적 갱신에 따른 무결성 유지가 무엇보다 중요하다 X
설명 : 데이터웨어하우스는 데이터를 실시간으로 갱신하거나 수정하지 않으며 
ETL 과정으로 주기적으로 적재된 데이터를 기반으로 리포팅만 수행하는 비휘발성 구조
데이터를 읽기 전용으로 유지하며, 트랜잭션 기반의 무결성 관리보다는 분석 가능한 
이력 유지에 초점을 둠 

* 빅데이터가 만들어내는 변화와 가장 거리가 먼 것은?
-빅데이터는 대량의 데이터를 다루기 때문에 정교한 표본조사 보다는 전체 데이터를
활용하는 경향이 강합니다. 따라서 이는 빅데이터의 변화와 거리가 멀다.

* 데이터 사이언스는 다양한 데이터에서 유용한 정보를 얻는 데 중점을 둔다. 적절하지
않은 것은?
- 주로 분석의 정확성에 초점을 두고 진행한다.
설명 : 데이터 사이언스는 분석의 정확성만을 목표로 하지 않는다. 
실무에서 데이터의 활용성, 실행 가능성, 확장성 등이 더욱 중요 할 수 있다. 

* 데이터 사이언스는 데이터 처리와 관련된 IT영역, 분석적 영역, 그리고 비즈니스 컨설팅 영역을
포괄하고 있다. 다른 영역은?
- 데이터 시각화 
설명 : 데이터 시각화는 비즈니스 컨설팅 영역에 속함, 데이터를 효과적으로 전달하고 인사이트를
도출하는 역학을 함 

* 미래 사회의 특성과 빅데이터 역할이 올바르게 연결되지 않은 것은?
- 단순화 - 경쟁력 
설명 : 단순화는 복잡한 것을 간단하게 만드는 것을 의미, 경쟁력과 직접적인 연관 없음
